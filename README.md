# WeatherApp
### Учебный проект
Регистрация и авторизация через Firebase. Добавленные города сохраняются Realtime, прогноз для них в Firestore.

---
### Библиотеки добавлены с помощью Swift Package Manager.
- Alamofire
- Kingfisher
- SwiftyJSON
- Firebase
